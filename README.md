# A Chat Game

  
用Unity开发的一个模拟微信聊天游戏，设计参考了[Emily is Away](http://emilyisaway.com/)（Unity） 和 [Coming Out Simulator 2014](http://ncase.me/cos/)（JavaScript，HTML/CSS），采用[Skech](https://www.sketchapp.com/)完成界面和原型设计。
